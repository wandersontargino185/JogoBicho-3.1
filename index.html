Como funciona o site

1. Cadastro de resultados

Digite um número de 0000 a 9999 e clique em "Adicionar". Os dados ficam salvos no seu navegador.

2. Estatísticas por posição

Para cada posição (Milhar, Centena, Dezena, Unidade) mostra frequência, porcentagem e atraso de cada dígito.

3. Palpites inteligentes

O site gera dois tipos de sugestão:

  • 🔥 Quentes: combina os dígitos que mais saíram em cada posição.
  • ❄️ Frios: combina os dígitos mais atrasados (que estão "devendo") em cada posição.

Quanto mais dados, melhores as sugestões.

4. Análise dos bichos

Frequência de cada grupo (bicho) baseado na dezena final.

5. Milhar repetida

Lista milhares que saíram mais de uma vez.

6. Importar/Exportar

Importe lista, exporte TXT, desfaça último, zere tudo.

Obs: Tudo fica salvo localmente. Limpar o cache apaga os dados.
